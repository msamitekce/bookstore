# Bookstore

## Endpoints
___

### REST

#### JSON ALL BOOKS
```
/books
```

#### JSON FIND BY ID

```
/restfind/{id}
```
*write a number instead of **{id}***

#### JSON DELETE BY ID

```
/restdelete/{id}
```
*write a number instead of **{id}***  
*returns "Success"*


### Views

#### ALL BOOKS
```
/index
```
or

```
/
```
*returns all books*

#### OTHER VIEWS

```
/add
```
and 

```
/edit
```
Views can be found on index page
